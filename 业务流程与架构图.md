# V1 轻量级项目全过程管理：业务流程与架构图

## 1. 项目全生命周期泳道图

```mermaid
flowchart LR
  subgraph A["申请人 / 项目负责人"]
    A1["新建项目并提交资料"]
    A2["修改退回项目"]
    A3["更新项目实施进度"]
    A4["提交验收与归档资料"]
  end
  subgraph B["项目审批人"]
    B1{"项目审批"}
  end
  subgraph C["采购经办人"]
    C1["登记采购"]
    C2["跟进采购并确定供应商"]
  end
  subgraph D["合同管理员"]
    D1["登记合同并跟踪履约"]
  end
  subgraph E["财务人员"]
    E1{"审核付款申请"}
    E2["登记付款时间"]
    E3["登记并核销发票"]
  end
  subgraph F["验收人员 / 归档审批人"]
    F1{"验收审核"}
    F2{"归档审批"}
  end

  A1 --> B1
  B1 -->|退回修改| A2
  A2 --> B1
  B1 -->|驳回| Z["项目结束"]
  B1 -->|通过，生成项目编号| C1
  C1 --> C2
  C2 -->|采购完成| D1
  D1 -->|履约中| A3
  A3 -->|按合同节点| E1
  E1 -->|驳回| A3
  E1 -->|通过| E2
  E2 --> E3
  E3 -->|实施完成| A4
  A4 --> F1
  F1 -->|整改 / 不通过| A3
  F1 -->|验收通过| F2
  F2 -->|退回补充资料| A4
  F2 -->|归档通过| Y["项目已归档"]
```

## 2. 项目管理操作泳道图

```mermaid
flowchart LR
  subgraph A["申请人 / 项目负责人"]
    A1["新建项目"] --> A2["填写基础信息、预算及附件"]
    A2 --> A3{"保存草稿或提交"}
    A4["按意见修改后再次提交"]
  end
  subgraph B["项目审批人"]
    B1["查看项目详情与附件"] --> B2{"审批处理"}
  end
  subgraph C["系统"]
    C1["校验必填项"]
    C2["生成正式项目编号并更新状态为已通过"]
  end
  A3 -->|保存草稿| C1
  A3 -->|提交审批| C1 --> B1
  B2 -->|退回修改| A4 --> C1
  B2 -->|驳回| D["更新状态为已驳回并记录意见"]
  B2 -->|同意| C2 --> E["进入采购管理"]
```

## 8. 项目归档操作泳道图

